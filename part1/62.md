## Python中用什么代替switch语句?

我想写一个函数,实现输入一个值对应输出另一个值.

在其他语言我可以用`switch`或者`case`语句,但是Python里没有`switch`.Python里如何解决这个问题?

***

你可以用一个字典:

```python
def f(x):
    return {
        'a': 1,
        'b': 2,
    }[x]
```

***

如果你希望设置一个默认值可以用字典的`get`方法:

```python
def f(x):
    return {
        'a': 1,
        'b': 2,
        }.get(x, 9)    # 9 is default if x not found
```
