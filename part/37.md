## 在 macOS 上如何安装 pip

昨天我花了许多时间研究怎么安装 `pip`, 但是没有找到方法

我该怎么安装它?

***

```python
easy_install pip
```

如果你需要管理员权限, 试一下:

```python
sudo easy_install pip
```
