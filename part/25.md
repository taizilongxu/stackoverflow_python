## 在 Python 里如何手工进行延迟?

***

```python
import time
time.sleep(5)   # Delays for 5 seconds. You can also use a float value.
```

这里有另一个例子展示每隔差不多一分钟运行一次:

```python
import time
while True:
    print("This prints once a minute.")
    time.sleep(60) # Delay for 1 minute (60 seconds).
```
